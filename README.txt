openMairie Exemple - ReadMe
===========================

openMairie Exemple est un framework permettant de montrer un exemple
d'utilisation de la librairie OpenMairie et de créer une application rapidement
permettant de générer des formulaires, des tableaux, des éditions PDF et
bien plus encore. 

Toutes les informations sur http://www.openmairie.org/


Documentation
-------------

Toute la documentation pour l'installation, l'utilisation et la prise en main
du framework dans le guide du développeur en ligne :
http://docs.openmairie.org/?project=framework&version=4.5&format=html
